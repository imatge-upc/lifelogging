The Visual Memory Game

This is a game for compute memorability score based on a crowdsorcing.

Instructions: When press 'start' button, many images will be displayed in the next 9 minutes. If you detect a second repetition of an image, press 'd'. When you finish the game, a text file will be download automatically to you computer, please attach this file to: marc.carne.herrera@estudiant.upc.edu

Please, be sure you are using "Chrome" browser.
