The Visual Memory Game

This is a game for compute memorability score based on a crowdsorcing.

Instructions: After enter the exercice assigned (between 1 and 5) press 'start' button. Many images will be displayed in the next 4:30 minutes. If you detect a second repetition of an image, press 'd'. When you finish the game, a text file will be download automatically to you computer, please attach this file to: marc.carne.herrera@estudiant.upc.edu

Please, be sure you are using "Chrome" browser.
